# Building LLM Agents

## Source Summary
Survey of LLM agent architectures, covering planning, tool use, memory, and feedback mechanisms. Surveys major agent systems including AutoGPT, Devin, Coze/LangChain, and MCP.

## Key Concepts
- ReAct (Reasoning + Acting)
- Chain-of-thought prompting
- Model Context Protocol (MCP)
- Agent reliability and evaluation

## Key Takeaways
1. LLM agents combine planning, tool use, memory, and feedback
2. Major open problems: reliability, cost, evaluation, safety
3. MCP enables tool interoperability across different agent frameworks
