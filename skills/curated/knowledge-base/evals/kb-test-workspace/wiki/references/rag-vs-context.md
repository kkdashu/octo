# RAG vs Long Context

## Source Summary
Comparison of Retrieval-Augmented Generation and long context windows for LLM applications. Recommends hybrid approach for medium-scale personal knowledge bases.

## Key Concepts
- Retrieval-Augmented Generation (RAG)
- Long context windows (100K+ tokens)
- Hybrid knowledge base approach
- Personal knowledge organization

## Key Takeaways
1. Long context wins when information is deeply interconnected or retrieval is impractical
2. RAG wins on cost, latency, and scalability for large corpora
3. For personal knowledge bases, quality of organization matters more than retrieval technique
