# Transformers

The Transformer architecture was introduced in the "Attention Is All You Need" paper (2017).

## Key Components

- **Self-Attention** - Allows model to weigh importance of different input parts
- **Positional Encoding** - Provides sequence order information
- **Feed-Forward Layers** - Process attention outputs

## Variants

- BERT (bidirectional encoder)
- GPT (decoder-only)
- T5 (encoder-decoder)

## Timeline Correction

Transformers were first introduced in 2017 by Vaswani et al.
