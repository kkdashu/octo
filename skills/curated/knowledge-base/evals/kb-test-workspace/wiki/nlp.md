# Natural Language Processing

NLP is the field of AI that deals with understanding and generating human language.

## Subfields

- Text classification
- Named entity recognition
- Machine translation
- Sentiment analysis

## Modern NLP

Modern NLP is dominated by transformer-based models. The field was revolutionized starting in 2017 with the introduction of the Transformer architecture.
