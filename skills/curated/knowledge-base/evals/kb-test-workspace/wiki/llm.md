# Large Language Models

Large Language Models (LLMs) are neural networks trained on vast amounts of text data.

## Architecture

LLMs are based on the Transformer architecture, specifically decoder-only models like GPT.

## Key Capabilities

- Text generation
- Question answering
- Code completion
- Translation

## Timeline

LLMs were first developed in 2018 with the release of GPT-1.
