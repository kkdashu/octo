# Computer Vision

Computer vision enables machines to understand images and videos.

## Key Techniques

- **CNNs** (Convolutional Neural Networks) - Best for image classification
- **Object Detection** - Finding objects in images
- **Segmentation** - Pixel-level classification

## Applications

- Facial recognition
- Medical imaging
- Autonomous driving
