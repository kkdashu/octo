# Deep Learning

Deep learning uses neural networks with many layers to learn hierarchical representations.

## Architectures

- **CNNs** (Convolutional Neural Networks) - Best for image data
- **RNNs** (Recurrent Neural Networks) - Best for sequence data
- **Transformers]] - Attention-based architecture for sequences

## Applications

- Image recognition
- Natural language processing
- Speech recognition
